# Empirical Risk Minimization Asymptotics

This repository contains a collection of scripts for defining statistical models and loss functions, and for solving Empirical Risk Minimization (ERM) problems both empirically and theoretically.

It accompanies our submission to **ICML 2026**, enabling the reproduction of experiments and the testing of new configurations.

## Repository Contents

*   **`Notebook1__fig_ICML2026.ipynb`**
    Reproduces the figures presented in the article.

*   **`Notebook2_simple_demo.ipynb`**
    A tutorial providing explanations on how to use the various classes and Python scripts included in this package.

*   **`Notebook3_universality_through_elastic_loss.ipynb`**
    Visualizes the performance universality breakdown when transitioning from ERM with $L_2$ loss to ERM with $L_1$ loss.
